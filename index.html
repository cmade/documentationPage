Javascript

Javascript is one of the essential languages of the web. Without Javascript, the web would not be interactive. Javascript allows control of a webpage through the use of the objects of its environment.

Your First Program

One of the first programs you should be a function that allows you to combine strings to say "Hello" + your name. This tends to be a popular program for beginers because it teaches you how to use Javascript a very basic level.

function greetMe(yourName) {
alert("Hello " + yourName);
}
greetMe("World");
